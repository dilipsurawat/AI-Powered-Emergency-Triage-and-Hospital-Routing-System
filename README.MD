## 📖 Overview

**AI-Powered Emergency Triage and Hospital Routing System** is an intelligent healthcare platform designed to assist patients during medical emergencies by combining Artificial Intelligence, geolocation, and real-time communication.

The system analyzes patient symptoms using **Google Gemini 3.5 Flash**, predicts the severity of the condition, recommends the most suitable nearby hospital, and enables quick access to emergency services through ambulance requests and direct communication with doctors via WhatsApp or phone call.

The application is built with a scalable backend using **FastAPI** and **Express.js**, a responsive frontend using **HTML, CSS, and JavaScript**, and **MongoDB** for secure data management.
